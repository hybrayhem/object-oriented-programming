Main provides a demo of all features of PegSolitaire class, just give user inputs.

Save and Load files must be formatted as follows:

0
--PPP--
--PPP--
PPPPPPP
PPP.PPP
PPPPPPP
--PPP--
--PPP--


First line is number of pegs out, don't touch. Initially zero.

Next lines are the board. Empty places must be filled by '-'. Only rectangular boards like above(including empty characters) can be loaded to game.
